## 🧠 Temas Abordados
- Os 4 pilares do Pensamento Computacional:
  - **Decomposição** — dividir problemas em partes menores
  - **Reconhecimento de padrões** — identificar semelhanças e repetições
  - **Abstração** — focar no essencial, ignorar o irrelevante
  - **Algoritmo** — criar passo a passo para resolver o problema
