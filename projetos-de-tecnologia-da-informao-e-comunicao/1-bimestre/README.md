## 🧠 Os 4 Pilares do Pensamento Computacional

1. **Decomposição** — Dividir um problema complexo em partes menores e mais fáceis de resolver
2. **Reconhecimento de Padrões** — Identificar semelhanças, tendências e regularidades
3. **Abstração** — Focar nas informações importantes, ignorando detalhes irrelevantes
4. **Algoritmo** — Criar uma sequência de passos para resolver o problema
