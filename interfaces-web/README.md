# 🌐 Interfaces Web

Atividades e projetos da disciplina de Interfaces Web, abordando HTML, CSS e fundamentos da Web.

## 📂 Conteúdo por Bimestre

### 1º Bimestre — [`1-Bimestre/`](./1-Bimestre/)
- Questionário sobre fundamentos da Web (HTTP, HTTPS, navegadores)
- Projeto: **Abril Laranja** — site sobre conscientização contra os maus tratos aos animais
- Projeto: **Origem do HTML** — história da Internet e do HTML

### 2º Bimestre — [`2-bimestre/`](./2-bimestre/)
- Projeto: **História do CSS** — site sobre a criação e evolução do CSS
- Projeto: **Site das Profissões** — site sobre profissões da área de TI
  - 🔗 [Ver online](https://site-profissoes.netlify.app/)

## 🧠 Temas Abordados
- Estrutura e semântica HTML
- Estilização com CSS
- Protocolo HTTP/HTTPS
- História e evolução da Web
- Publicação de sites (Netlify)
